# Voltaras AI/ML Backend

## Overview
This project is a scalable backend system built using FastAPI to support AI/ML workflows and data processing.

## Tech Stack
- Python
- FastAPI
- REST APIs
- Google Cloud (BigQuery, Cloud Storage)

## Features
- REST API endpoints for data handling
- Cloud-based data processing
- Scalable backend architecture

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the FastAPI server
